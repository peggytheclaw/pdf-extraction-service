üöÄ REST API Overview

Our RESTful API provides programmatic access to PDF extraction capabilities. Upload documents, retrieve extraction results, and manage jobs through standard HTTP requests.

Base URL

https://api.pdfextraction.service/v1

Authentication

All API requests require authentication via API key in the Authorization header:

Authorization: Bearer YOUR_API_KEY_HERE

Obtain your API key from the dashboard after signing up. Keep your API key secure‚Äîdo not expose it in client-side code or public repositories.

Core Endpoints

POST /extract

Upload and extract data from a PDF document

Request Body (multipart/form-data)

{
  "file": <PDF binary data>,
  "extraction_template": "invoice|receipt|form|custom",
  "output_format": "json|csv|xml",
  "webhook_url": "https://your-domain.com/webhook" (optional)
}

Response (202 Accepted)

{
  "job_id": "job_abc123xyz",
  "status": "processing",
  "estimated_completion": "2024-02-09T14:30:00Z",
  "webhook_enabled": true
}
GET /extract/{job_id}

Check status and retrieve extraction results

Response (200 OK - Completed)

{
  "job_id": "job_abc123xyz",
  "status": "completed",
  "created_at": "2024-02-09T14:25:00Z",
  "completed_at": "2024-02-09T14:25:47Z",
  "processing_time_ms": 47200,
  "data": {
    // Extracted data in requested format
  },
  "confidence_score": 0.998,
  "pages_processed": 3
}
DELETE /extract/{job_id}

Delete extraction job and associated data

Response (204 No Content)

Job and all associated data permanently deleted.

Example: Complete Workflow

# 1. Upload document for extraction
curl -X POST https://api.pdfextraction.service/v1/extract \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -F "file=@invoice.pdf" \
  -F "extraction_template=invoice" \
  -F "output_format=json"

# Response: {"job_id": "job_abc123xyz", "status": "processing"}

# 2. Check extraction status (poll every 5-10 seconds)
curl -X GET https://api.pdfextraction.service/v1/extract/job_abc123xyz \
  -H "Authorization: Bearer YOUR_API_KEY"

# Response when complete:
# {
#   "status": "completed",
#   "data": {
#     "invoice_number": "INV-2024-00892",
#     "total": 5573.37,
#     ...
#   }
# }

X-PDF-Extraction-Signature: sha256=...

{
  "job_id": "job_abc123xyz",
  "status": "completed",

üîí Webhook Security

Verify webhook authenticity using the X-PDF-Extraction-Signature header. Calculate HMAC-SHA256 of the request body using your webhook secret.

    template='invoice'
)

npm install @pdfextract/sdk

const PDFExtract = require('@pdfextract/sdk');

const client = new PDFExtract('YOUR_KEY');
const result = await client.extract({
  file: 'invoice.pdf',
  template: 'invoice'
});
